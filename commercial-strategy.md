### 路径A：SaaS平台 — 烟草/医药监管团队订阅

| 维度 | 评估 |
|------|------|
| **年营收潜力（Y1）** | $5K-$30K（3-10个早期客户，$100-300/月） |
| **年营收潜力（Y3）** | $100K-$500K（50-100客户，含企业级客户） |
| **每周投入** | 15-25小时（开发+运维+客服+功能迭代） |
| **创始人匹配度** | **低** — 需要持续产品开发、客户支持、GTM、合规（GDPR/SOC2等），大幅超出副项目能力 |
| **风险** | 高 — 长销售周期（烟草行业采购保守）、强竞争对手(Euromonitor, IQVIA)、需要大量资本 |
| **首步行动** | 建landing page + waitlist，用LinkedIn/Twitter内容吸引自然注册 |

**评分：2/10** — 作为副项目不可行，需全职投入和融资。

### 路径B：数据即服务(DaaS) — 按报告定价PMTA/MRTP证据包

| 维度 | 评估 |
|------|------|
| **年营收潜力（Y1）** | $10K-$50K（2-5份定制报告，$5K-$15K/份） |
| **年营收潜力（Y3）** | $100K-$300K（10-20份报告/年 + 数据订阅） |
| **每周投入** | 5-10小时（模板化后大部分可复用） |
| **创始人匹配度** | **中高** — 利用现有分析能力，模板化后边际成本低，不需陌生拜访（客户通过方法论白皮书/仪表板找来） |
| **风险** | 中 — 客户数量有限（美国烟草PMTA申请者约20-30家），需证明"not just public data" |
| **首步行动** | 制作1份样本PMTA Evidence Package（以Geek Bar为案例），PDF格式，发布在网站上作为lead magnet |

**评分：7/10** — 高杠杆、高匹配度，但市场容量有天花板。

### 推荐方案：Top 2 路径组合

## 推荐路径 1: 内容/研究变现（路径C） — 主线

**理由：**
- 完美匹配创始人条件（写分析=核心能力，异步工作，无需社交）
- 现有资产（NYTS 2025分析、方法论白皮书、Geek Bar简报）即可作为启动内容
- 零成本启动
- 三方交叉验证的独特性是极强的内容护城河

## 推荐路径 2: 数据即服务（路径B） — 高价值补充

**理由：**
- 利用Newsletter建立的受众，自然转化为付费报告客户
- 单价高（$5K-$15K/份），少量客户即可产生有意义的收入
- PMTA Evidence Package是高度标准化的可模板化产品
- 客户主动找来（通过方法论白皮书和仪表板），不需要outbound
